# Assets

Put all website images in `src/assets`.

## New upload folders

Create/keep images in these folders so the UI sections can reference them consistently:

- `src/assets/services/`
  - `mortuary/`
  - `decorations/`
  - `fleet/`

- `src/assets/testimonials/`

- `src/assets/funerals/weekly/`

## Naming convention

Use lowercase, hyphen-separated names.

Examples:
- `src/assets/services/mortuary/viewing-area-01.jpg`
- `src/assets/services/decorations/decor-01.jpg`
- `src/assets/services/fleet/hearse-01.jpg`
- `src/assets/testimonials/testimonial-tayo-01.jpg`
- `src/assets/funerals/weekly/sivuyile-lwana-01.jpg`

## Notes

- Prefer JPG for photos.
- Keep images under ~500KB when possible (compress before uploading).
